El método delete también es muy sencillo de implementar con lo que hemos visto. Este método lo que hace es borrar un determinado mooc que llega como parámetro.

En esta tarea vamos a completar ese método con lo que falta, que en este caso sería localizar el mooc en model.Moocs y proceder a su borrado si lo encontramos. Una vez borrado añadir una traza de log indicando el ID del mooc eliminado, ¿podrías en vez de el ID indicar el name de ese mooc en la traza? 

mooc = model.Mooc.get_by_id(int(mooc_id))
if mooc is None:
    send(self, 404)
else:
    name = mooc.name # Cogemos el nombre del mooc antes de borrar para luego mostrarlo en la traza
    mooc.key.delete()
    # Dos maneras de mostrar mensajes
    logging.info('WNP: Se procede a borrar el mooc: ' + format(mooc_id) + ' - ' + name)
    logging.info('WNP: Se procede a borrar el mooc con id %s y nombre %s', format(mooc_id), name)
    send(self, 204)
