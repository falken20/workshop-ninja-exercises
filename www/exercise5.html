Instalar dependencias

Dentro del raíz del proyecto tenemos el fichero requirements.txt, este fichero contiene las dependencias de librerías Python que vamos a necesitar en nuestra aplicación.

Si lo abrimos veremos que en nuestro caso sólo hay un paquete que es para el acceso a Cloud Storage. Para instalar estas dependencias debemos ejecutar el siguiente comando donde le indicaremos el fichero de dependencias así como la carpeta donde se instalaran (lib): 

sudo pip install -t lib -r requirements.txt

Validar que no devuelve ningún mensaje de error y que crea la carpeta lib en nuestro proyecto
