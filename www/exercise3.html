Entorno virtual

Python dispone de una herramienta llamada virtualenv que permite utilizar diferentes versiones de Python y de las librerías necesarias en cada proyecto, mediante la creación de entornos virtuales.

Para crear nuestro entorno virtual Python, en la carpeta del proyecto vamos a ejecutar el siguiente comando donde indicamos que se cree un entorno virtual para Python2 y en la carpeta /env. 

virtualenv -p python2 env
Validar que tenemos creada la carpeta env dentro del proyecto
