./src/moocs.py

En este fichero se encuentra la clase y los métodos necesarios para interactuar con la entidad model.Mooc, es decir, la que va a gestionar los moocs que se den de alta para cada ninja.

Tiene los mismos métodos que la clase Ninjas excepto update ya que los moocs no se pueden modificar.

Es decir, contiene:

  • el método list para obtener los moocs
  • el método retrieve para obtener los datos de un mooc
  • el método create para crear un mooc
  • y el método delete para eliminar un mooc.

Para empezar vamos a intentar realizar el metodo list entero, no te preocupes porque es muy parecido al metodo list del fichero ninjas.py. Lo único es que debes tener en cuenta que aquí debes obtener el ID del ninja y con ese dato listar los moocs que le pertenecen, por tanto deberías hacer consultas sobre model.Mooc. 

ninja_id = self.request.get('ninja_id', default_value=None)
if ninja_id is None:
    moocs = model.Mooc.query().order(model.Mooc.date).fetch()
else:
    moocs = model.Mooc.query(model.Mooc.ninja_id == int(ninja_id)).order(model.Mooc.date).fetch()
