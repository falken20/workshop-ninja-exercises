app.yaml

Volvemos al fichero app.yaml, el elemento handlers es un elemento obligatorio en el archivo.

El elemento proporciona una lista de patrones de URL y descripciones acerca de cómo deben manejarse.

En este fichero tambien debemos especificar el entorno de ejecución que usaremos para Python así como la versión del API de este entorno de ejecución. Ambos parámetros también son obligatorios. Localizar ambos parametros.

Los encontrarás al principio del archivo

runtime: python27
api_version: 1
