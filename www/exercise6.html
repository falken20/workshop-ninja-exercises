RESUMEN DEL BLOQUE

  • Hemos clonado los fuentes del proyecto en nuestro equipo
    • git clone https://github.com/richionline/Workshop-Ninja-Python.git
  • Hemos accedido a la consola Google en la nube para crearnos un segmento en Storage, que recuerda será donde se guardaran los archivos usados por la aplicación
  • Hemos creado y probado un entorno virtual basado en Python 2 para poder ejecutar en local nuestra aplicación
    • virtualenv -p python2 env
    source env/bin/activate
  • Hemos instalado en la carpeta lib en local las dependencias necesarias por nuestro proyecto 
    • sudo pip install -t lib -r requirements.txt
