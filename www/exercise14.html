appengine_config.py

El archivo appengine_config.py se usa para poder personalizar los módulos y APIs de Python en Google App Engine. Debe estar en el raíz de nuestro proyecto. Con ello podemos anular (sobreescribir) constantes y funciones existentes en estos módulos. Para anular una constante o función, define la función o constante con el nombre del módulo, un guión bajo y el nombre de la constante o función a anular.

En este caso vamos a anular la función default_namespace_for_request del modulo namespace_manager para que devuelva como namespace el que hemos definido en nuestro fichero config.py. Con esto conseguimos que cuando nuestro servicio reciba una petición HTTP, utilice el espacio de nombres correspondiente al nombre del mismo.

Modificar el método namespace_manager_default_namespace_for_request() para que devuelva la constante definida en config.py, para ello usar el comando return valor. Como pista, para referenciar una variable Python de otro fichero podemos hacerlo de la forma "fichero.variable". 

def namespace_manager_default_namespace_for_request():
    return config.NAME_SPACE
