Entorno virtual
Ahora vamos a comprobar que lo hemos creado correctamente, para ello vamos a arrancar el entorno virtual con el comando 
source env/bin/activate
Debería aparecernos en la línea de comandos algo que identifica que hemos arrancado correctamente el entorno virtual 
    ~/...../workshop-ninja-python $ source env/bin/activate
    (env) ~/...../workshop-ninja-python $
El prefijo (env) nos indica que hemos arrancado el entorno virtual
