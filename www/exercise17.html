En este fichero Python tenemos la definición de las entidades (tablas) de Data Store que vamos a usar, cada una correspondiente a una clase con sus respectivos atributos (columnas).

Para ello usamos la librería NDB, que está referenciada en un import al principio del fichero. Esta librería permite que las aplicaciones Python en App Engine se conecten a Cloud Data Store de manera muy sencilla.

Si observamos el fichero vemos que tenemos dos entidades definidas, junto con todos sus campos así como el tipo de cada campo. Revisa estas definiciones y detecta qué campos son índices. Aparte de revisar lo comentado incorpora como índices los siguientes campos para ello tendrás que marcar como True su priopiedad indexed:

  • En clase Ninja incorpora como índices los campos name y department
  • En clase Mooc incorpora com índice el campo date

    name = ndb.StringProperty(required=True, )
    department = ndb.StringProperty(required=True, )
...
    date = ndb.DateTimeProperty(auto_now=True, )
